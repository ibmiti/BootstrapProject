WHAT: A Blog

My greatest tool for learning has not been coding tutorials nor boot camps it has been the question of "What?" for example: when Building a new project I ask myself 'What' is required for the build, or when I run into a concept that boggles me, I quickly ask myself 'What''s the key concept behind which makes it function, 'Whats' the 'use-case' 'What' are the components that make up the whole. +++++++++++++++++++++++++++++++ Being a programmer for me has been a journey of questions with eventual answer's, and the most powerful question of them all, for me at-least has been What is it? and after I can define the problem I can Solve the problem
